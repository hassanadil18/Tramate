# Discord User Connection Guide

## Overview
This guide walks you through connecting your Discord account to access premium trading channels on the Tramate platform.

## 🚀 Quick Start

### Step 1: Join the Discord Server
Before you can connect to any channel, you must first join the Discord server:

1. Click the "Join Discord Server" button on any channel page
2. Accept the Discord invite
3. Make sure you have the required role for the channel you want to access

### Step 2: Connect to a Channel
1. **Navigate to Channels**: Go to the "Channels" section in your dashboard
2. **Select a Channel**: Choose the premium channel you want to access
3. **Click "Verify & Connect"**: This will start the verification process

### Step 3: Discord Username Verification
1. **Enter Your Discord Username**: Use any of these formats:
   - `username#1234` (legacy format with discriminator)
   - `@username` (new Discord format)
   - `username` (basic format)

2. **Click "Verify Membership"**: The system will check if you're a member of the Discord server

3. **Wait for Verification**: Usually takes 2-5 seconds

## ✅ Successful Connection

When verification succeeds, you'll see:
- ✅ A green success popup with confirmation
- ✅ Automatic connection to the channel
- ✅ Redirect to your channels dashboard
- ✅ Access to trading signals from that channel

## ❌ Common Issues & Solutions

### "Username not found in server"
**Problem**: Your Discord username wasn't found in the server member list.

**Solutions**:
1. **Make sure you've joined the server**: Click the Discord invite link first
2. **Wait 2-3 minutes**: Discord membership can take time to sync
3. **Check your username**: Make sure it's spelled exactly right
4. **Try different formats**: Use `username#1234`, `@username`, or just `username`

### "Invalid Discord username format"
**Problem**: The username format isn't recognized.

**Solutions**:
1. **Check for typos**: Make sure there are no extra spaces or characters
2. **Use the correct format**: 
   - Include the # and 4 numbers if you have them
   - Or use the new @username format
   - Or just the username without @ or #

### "You don't have the required role"
**Problem**: You're in the server but don't have access to this specific channel.

**Solutions**:
1. **Check your subscription**: Make sure your subscription includes this channel
2. **Contact the server admin**: Ask them to assign the correct role
3. **Verify your payment**: Ensure your subscription is active

### "Connection session expired"
**Problem**: The verification process took too long.

**Solutions**:
1. **Start over**: Go back to the channels page and try again
2. **Clear your browser**: Refresh the page and start the process again

## 🔐 How Verification Works

1. **Username Input**: You enter your Discord username
2. **API Check**: Our system calls Discord's API to check server membership
3. **Role Verification**: We verify you have the required role for the channel
4. **Access Grant**: If successful, we create a connection between your accounts
5. **Signal Access**: You now receive trading signals from that Discord channel

## 🎯 Best Practices

### For Smooth Connections:
1. **Join the Discord server first**: Always join before trying to connect
2. **Use your exact username**: Copy/paste to avoid typos
3. **Wait for sync**: Give Discord a few minutes after joining

1. **Connect one at a time**: Don't rush the process
2. **Verify each connection**: Make sure each one works before moving to the next
3. **Update Discord roles**: When your subscription changes, update your Discord roles
